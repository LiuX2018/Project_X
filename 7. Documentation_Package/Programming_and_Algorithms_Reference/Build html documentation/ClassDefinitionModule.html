Class Definition Module

It is a module of the toolbox where all the class definitions are placed. Each class is defined with its own properties and methods. The properties are all listed inside the class definition file and all the methods are placed inside a folder named "@Class_Name". Defining the methods in a separate file reduces the size of the main class definition file and makes it more easy to read.


The toolbox has two main categories of classes :

  1. GUI Related
  2. Non GUI Related
