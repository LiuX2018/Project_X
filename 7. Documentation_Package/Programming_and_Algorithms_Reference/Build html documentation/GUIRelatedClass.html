GUI Related Class Definitions

These are classes that are used for building and operating the graphical user interface of the toolbox. Basically there are two classes used for the GUI.

  1. AODParentWindow:  Class for the Main GUI of the toolbox.
  2. AODChildWindow: Class for all child windows which pop up on the top of the main window.


Design Philosophy

  • All GUIs are built directly by writing the MATLAB codes to define, position and set properties of the uicontrols with out using GUIDE. This allowed design of more dynamic and easily programmable user interfaces. There will be no .FIG file for the user interfaces and the number of files required for a GUI will be reduced from 2 to 1.
