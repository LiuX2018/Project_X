plotDelanoDiagram

PURPOSE

plotDelanoDiagram: Plots Y-Ybar (Delano) diagram for a given optical system.

SYNOPSIS

function [ output_args ] = plotDelanoDiagram( input_args )

DESCRIPTION 

 plotDelanoDiagram:  Plots Y-Ybar (Delano) diagram for a given optical system.
 The Y-Ybar diagram is a plot of marginal ray (in Y) height as a function  
 of chief ray (in X) height for a paraxial skew ray at every surface in the lens
